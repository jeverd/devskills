# Skill template

Copy this folder and edit SKILL.md for your own skill.

## When to use

- When you want to create a new skill for devskills

## Steps

1. **Copy this folder** to the right category under `skills/`
2. **Fill in the frontmatter** — name, description, category, tags
3. **Write the skill** following the sections below
4. **Add references/scripts/templates** in subfolders if needed
5. **Test it** — follow your own instructions from scratch
6. **Submit a PR**

## Pitfalls

- Don't skip the frontmatter — it's how skills get indexed
- Don't make skills too broad — one focused skill beats a vague mega-skill
- Don't forget pitfalls — they're often the most valuable part

## Verification

- Someone unfamiliar with the topic can follow the skill and succeed
- The skill has a clear trigger (when to use it)
- Pitfalls cover at least one non-obvious gotcha
